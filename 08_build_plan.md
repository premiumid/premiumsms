# Build Plan

## Phase 1

- Set up Next.js, Supabase, and auth.
- Build service and country catalog.
- Create wallet and rental tables.

## Phase 2

- Integrate one SMS provider.
- Build rental creation and SMS inbox.
- Add provider webhooks.

## Phase 3

- Add payment topups.
- Add admin tools.
- Add refund and expiration jobs.

## Phase 4

- Improve UI polish.
- Add API keys for developers.
- Add analytics and monitoring.
- Add localization, including Kinyarwanda.

## Deliverables

- MVP web app.
- Admin panel.
- Provider adapter layer.
- Documentation.
- API reference.
